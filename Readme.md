# Jorge Gimenez Garcia - Senior Data Engineer Portfolio 🚀

Welcome to the source code of my professional portfolio! This repository hosts my personal website, showcasing my experience, technical skills, and projects as a Senior Data Engineer.

🌐 **Live Website:** [jorgegimenezgarcia.github.io](https://jorgegimenezgarcia.github.io)

## 🏗️ Architecture & Stack

To ensure maximum performance, instant load times, and a highly maintainable codebase on GitHub Pages, this portfolio is built using modern front-end standards:

* **HTML5:** Semantic structure ensuring accessibility.
* **Tailwind CSS (via CDN):** Utility-first CSS framework for a responsive, clean, and corporate UI layout without the overhead of massive custom stylesheets.
* **Vanilla CSS3 (`style.css`):** Scoped explicitly for custom scrollbar configurations and hardware-accelerated animations (Fade-in).
* **Vanilla JavaScript:** Lightweight Intersection Observer implementation to handle scroll-based view animations dynamically.

* **AI / Machine Learning:** NLP, TensorFlow, Keras, Recurrent Neural Networks (RNNs).

## 🚀 How to deploy or update

This repository is connected to **GitHub Pages**. Any changes pushed to the `main` branch will automatically update the live site.

1.  **Edit Content:** Modify the `index.html` file to update work experience or skills. Tailwind classes handle the layout.
2.  **Update Resume (PDF):** Replace the `Jorge_Gimenez_Garcia_CV.pdf` file in the root directory with a new version when needed. The website's "Download PDF" button links directly to this file automatically.
